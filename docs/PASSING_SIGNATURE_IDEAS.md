# Passing Signature Ideas

This is a future Phase 6/7 research note, not Phase 5 implementation.

Potential PassSonar-style features:

- pass angle bins
- pass length bins
- forward/lateral/backward share
- switch-field share
- central-to-wide progression
- pass direction entropy
- team passing fingerprint

These features should be built from measured event locations and pass endpoints. They should not become identity claims until tested against real data coverage and backtests.
