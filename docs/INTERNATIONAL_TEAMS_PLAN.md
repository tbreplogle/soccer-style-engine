# International Teams Plan

International soccer should be supported separately from club-league projections.

National teams have sparse match samples, neutral sites, tournament formats, changing rosters, uneven opponent quality, and long gaps between meaningful fixtures. Club league ratings should not be mixed with country ratings.

## Phase 11 Audit

The international-readiness audit inspects local StatsBomb Open Data metadata if present. It lists international competitions such as World Cup, Euros, Copa America, AFCON, and Women's World Cup when they exist locally.

The audit reports:

- available international competitions
- match counts by competition and season
- whether event data is present
- whether 360 data is present
- whether current live international data exists locally

It does not build international score projections.

## Future International Phase

A future international module should add:

- sparse-sample priors
- neutral-site handling
- tournament-stage context
- opponent-strength normalization
- roster and manager volatility notes
- strict separation between country and club ratings
