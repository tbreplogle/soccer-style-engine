# International Data Limitations

International soccer needs softer assumptions than club leagues.

Key limitations:

- national teams play fewer matches
- neutral sites are common
- tournament hosts can have a separate advantage
- rosters and managers change between windows
- friendlies and tournament matches have different incentives
- opponent strength is uneven
- current live international event data is not available by default

Historical StatsBomb Open Data can validate event-style context for tournaments, but it should not be described as current live data.

When neutral-site status is unknown, projections add uncertainty flags and use reduced home advantage.
