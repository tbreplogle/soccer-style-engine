# Phase 21 World Cup Data Stack Example

World Cup projection candidate stack:

```text
sofascore -> eloratings -> fbref -> manual_fallback -> statsbomb_open_data
```

SofaScore is a candidate for current fixtures/stats. EloRatings is a candidate for national-team strength. FBref may provide aggregate competition context. Manual fallback remains important when free source coverage is incomplete.

StatsBomb Open Data is historical validation only, not a current World Cup data feed.
