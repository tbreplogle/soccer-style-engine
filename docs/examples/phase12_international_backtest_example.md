# Phase 12 International Backtest Example

This is an example format, not a generated report.

Expected backtest summary fields:

- matches evaluated
- Team A goals MAE
- Team B goals MAE
- total goals MAE
- W/D/L log loss
- Brier score
- exact score hit rate
- over/under 2.5 accuracy
- sparse-sample warning rate
- confidence bucket performance

Use the backtest to learn whether the foundation is stable enough for the next international phase. Do not treat early international outputs as a picks product.
