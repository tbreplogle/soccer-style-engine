# Phase 22 Manual Fallback Example

Manual fallback file:

```text
data/sample/current_international_matchups.csv
```

Required columns:

- `match_date`
- `competition`
- `round_name`
- `group_name`
- `home_team`
- `away_team`
- `neutral_site`

Manual rows are committed sample inputs. Generated slate and projection outputs remain ignored.
