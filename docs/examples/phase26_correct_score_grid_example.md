# Phase 26 Correct Score Grid Example

The correct score grid is organized by away goals on rows and home goals on columns.

```text
Away \ Home | 0       | 1       | 2
0           | 8.1%    | 11.5%   | 8.2%
1           | 7.7%    | 10.9%   | 7.8%
2           | 3.7%    | 5.2%    | 3.7%
```

When available, each cell also shows model-implied American odds under the probability. Top correct scores are listed separately above the grid.

Score labels use spaces around the dash, such as `1 - 0`, so spreadsheet software does not treat the score as a date.
