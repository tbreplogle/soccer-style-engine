# Phase 13 Projection Profile Comparison Example

This is an example output shape, not a generated report.

Profile comparison rows show each profile side by side:

- xG
- projected total
- most likely score
- W/D/L probabilities
- confidence
- market influence for club projections
- warnings
- profile disagreement flags

Disagreement flags help identify unstable matchups. They are diagnostic context, not betting guidance.
