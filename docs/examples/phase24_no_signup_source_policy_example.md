# Phase 24 No-Signup Source Policy Example

Allowed in the Phase 24 default backbone:

- OpenFootball/football.db-style static fixtures
- EloRatings-style local/sample ratings
- manual fallback CSV
- ESPN scoreboard only as optional fallback

Excluded:

- TheStatsAPI
- API-Football
- Sportmonks
- API-key feeds
- signup-required feeds
- paid APIs
- current StatsBomb
