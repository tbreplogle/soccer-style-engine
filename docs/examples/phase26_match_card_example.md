# Phase 26 Match Card Example

A board match card is intended to make one projection row readable without opening CSV files.

Example fields:

```text
Curacao vs Cote d'Ivoire
Projected xG: Curacao 1.421 | Cote d'Ivoire 0.947
Projected total: 2.368
Most likely score: 1 - 0
Support: low_manual_fixture_rating
Confidence: low
Source tier: manual
Style inputs: unavailable
Rating: rating_only
```

The card then shows 1X2 probability output, totals, BTTS, clean sheet probabilities, top correct scores, and the correct score grid. Manual rows remain labeled as user supplied and not source-verified.
